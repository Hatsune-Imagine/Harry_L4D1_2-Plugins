# 中文說明
當有人在背後驚嚇Witch，Witch會秒轉身攻擊

* 原理
    * 在官方預設當中，有人從背後某一個角度驚嚇Witch，會讓Witch卡住 (請看影片展示)
    * 安裝這個插件之後，有人從背後驚嚇Witch，幫Witch自動轉身
