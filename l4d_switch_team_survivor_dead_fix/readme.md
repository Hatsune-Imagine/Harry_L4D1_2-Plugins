# 中文說明
修復切換陣營之後會導致倖存者死亡或倒地的Bug (官方的bug)

> __Note__ 此插件已停止更新, 請使用 [emove_touch_links](https://github.com/shqke/sp_public/tree/master/remove_touch_links)

* 原理
    * 如何重現bug
        1. 特感站在人類會死掉的區域 (譬如河邊、懸崖、屋頂)
        2. 特感按M切換陣營到倖存者
        3. 倖存者會死亡或倒地
