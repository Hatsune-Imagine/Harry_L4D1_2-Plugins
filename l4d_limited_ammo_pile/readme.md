# 中文說明
子彈堆只能拿一次子彈，當每個人都拿過一遍之後移除子彈堆

> __Note__ 
<br/>此插件已停止更新，請使用[Percentage Limited Ammo Pile](https://forums.alliedmods.net/showthread.php?t=340484)

* 原理
	* 子彈堆拿一次之後就不能再拿了
	* 每個人都拿過一遍之後，子彈堆自動移除
